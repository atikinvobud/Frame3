# Проект
Проект является распределённым монолитом.

Бэкенд написан на ASP.NET Core, в качестве баз данных используются PostgreSQL и Redis.

Фронтенд написан на Vue3.

# Установка

## Клонировать репозиторий.
```sh
git clone https://github.com/atikinvobud/Frame3
````

## Для запуска проекта в среде разработки:

Первый терминал (Фронтенд):
````sh
cd Frontend
npm install
npm run dev
````
Второй терминал (Бэкенд):
````sh
cd Backend
dotnet run
````

Проект будет доступен по адрессу:
````sh
http://localhost:5173/
````

## Для запуска в среде продакшена:

````sh
docker-compose up -d
````

Проект будет доступен по адрессу:
````sh
http://localhost:5173/
````
